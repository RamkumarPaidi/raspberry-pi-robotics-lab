# 🤖 Raspberry Pi Robotics Lab

A hands-on robotics learning journey using Raspberry Pi.

This repository is designed for beginners who want to learn robotics step by step by building real working systems.

---

# 🎯 Goal

To learn robotics by building real systems using Raspberry Pi.

You will learn:
- How sensors help robots see the world
- How motors bring movement to life
- How robots make decisions
- How to build autonomous robots step by step

---

# 🧠 Learning Style

This is a build-first learning lab.

Instead of heavy theory, you will:
- Write small code daily
- Test hardware behavior
- Build real robotics projects step by step

---

# 🧭 Learning Roadmap

## 00 - Introduction
Basic understanding of robotics and Raspberry Pi

## 01 - GPIO Basics
Learn how to control LEDs, buttons, and signals

## 02 - Sensors
Learn how robots detect the world

## 03 - Motors
Learn how robots move

## 04 - Robot Logic
Learn decision making for robots

## 05 - Computer Vision
Teach robots to see

## 06 - Autonomous Robots
Build self-moving robots

## 10 - Mini Projects
Real-world robotics builds
